1. Veritabanı: SQLite
Teknoloji: SQLite (hafif, dosya tabanlı veritabanı)
Dosya Konumu: backend/database/appointment_system.db
Avantajlar:
Kurulum gerektirmez
Tek dosya
Hızlı ve güvenilir
Geliştirme için ideal
----------------------------------------------------------
2. Tablolar ve Veri Saklama:

* SQL Users Tablosu

- id: Otomatik artan benzersiz ID
- email: Kullanıcı e-postası (benzersiz)
- password_hash: Şifrelenmiş parola (bcrypt)
- full_name: Tam ad
- user_type: 'consultant' veya 'client'
- is_verified: Doğrulama durumu
- phone: Telefon numarası
- created_at: Kayıt tarihi

*SQL Appointments Tablosu

- id: Otomatik artan benzersiz ID
- consultant_id: Danışman ID (users tablosuna bağlı)
- client_id: Müşteri ID (users tablosuna bağlı)
- start_time: Randevu başlangıç zamanı
- end_time: Randevu bitiş zamanı
- status: 'pending', 'confirmed', 'cancelled', 'completed'
- notes: Randevu notları
- created_at: Oluşturulma tarihi
- updated_at: Güncellenme tarihi

*SQL Notifications Tablosu

- id: Otomatik artan benzersiz ID
- user_id: Kullanıcı ID (users tablosuna bağlı)
- title: Bildirim başlığı
- message: Bildirim mesajı
- type: 'appointment', 'system', 'reminder'
- is_read: Okunma durumu
- created_at: Oluşturulma tarihi

3. Güvenlik ve Şifreleme
🔐 Parola Güvenliği
bcryptjs kullanılıyor
12 salt rounds ile şifreleme
Parolalar asla açık metin olarak saklanmıyor
🔑 JWT Token Sistemi
Kullanıcı girişi sonrası JWT token oluşturuluyor
Token'lar HTTP-only cookie'de saklanıyor
Her API isteğinde token doğrulanıyor
